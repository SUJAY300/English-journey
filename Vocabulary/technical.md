# Technical Vocabulary

## Scalability

### Meaning

The ability of a system to handle increasing load efficiently.

### Example

Our backend architecture is designed for scalability.

### My Sentence

I want to learn system design to build scalable applications.

### Related Words

* Reliability
* Performance
* Throughput

---

## API

### Meaning

A set of rules that allows applications to communicate.

### Example

The frontend communicates with the backend through APIs.

### My Sentence

I created a REST API for a tax calculator project.

### Related Words

* Endpoint
* Request
* Response
* REST

---

## Stakeholder

### Meaning

A person or group affected by a project.

### Example

The product manager is an important stakeholder.

### My Sentence

Backend developers often communicate with stakeholders to understand requirements.
